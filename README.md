# 소개

안녕하세요. **iOS 중심(메인) 모바일 개발자**입니다.  
주력은 **Swift**이며, 제품 관점에서 **사용자 경험(UI/UX)과 안정성**을 중요하게 생각합니다.

## 기술 스택
- **iOS (Main)**: Swift, SwiftUI, Combine, MVVM, UIKit
- **Android (Sub)**: Kotlin, Java  
  - 안드로이드는 메인보다는 **서브로 지원/협업** 가능합니다.

## 관심사
- 사용자에게 “매일 쓰게 되는” **실용적인 앱** 만들기
- 아키텍처(MVVM), 상태 관리, 데이터 저장/동기화, 디버깅/안정화
- 미니멀한 UI와 Apple HIG 기반의 완성도 높은 UX
